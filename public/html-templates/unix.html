Unix/Command Line

Note: The code for the commands is provided in the outlined boxes. The $ is a part of the command prompt. The code is the text following the $

Code inside of these brackets: [ ] is to be replaced by your respective file names

Unix Basics

Command Line Arguments

To list the files in the current directory use the following command

$ ls

We can simply change the directory we are in by using the following command

$ cd

Modifying Files and File Structure through CLI

If we want to move a file to another directory, we can use the following command line argument

$ mv

To create a new folder for a project, use the following command line argument

$ mkdir [name]

To remove a directory use the following

$ rmdir [name]

To create a new file in the current directory use the following command

$ touch [file.txt]

To remove a certain file from the current directory use the following command

$ rm [file.txt]

If you want to have a clear view of the terminal and not see the previous commands entered, you can use the following command to clear the terminal window

$ clear
